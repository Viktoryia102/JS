1. Kiedy powstał JavaScript
    Język skryptowy został stworzony w 1995 roku

  2. Dlaczego jest to język skryptowy
    Jest interpretowany – nie musi być kompilowany, wykonuje go przeglądarka.

  3. Gdzie jest wykonywany JS
    W przeglądarce, która obsługuje JS

  4. Dlaczego jest JS to bezpieczne narzędzie
    JavaScript ma znacznie ograniczone uprawnienia dostępu do zasobów komputera, przy użyciu którego przeglądana jest dana strona, a wszelkie odwołania do funkcji i obiektów wykonywane są w trakcie wykonywania programu.

  5. Czy jest to język obiektowy , jeśli tak to dlaczego.
    Tak, JavaScript jest językiem obiektowym, ponieważ wykorzystuje obiekty, metody, zdarzenia i opiera się na strukturze DOM (Document Object Model). Pozwala na tworzenie i modyfikowanie obiektów.

  6. Wymień zalety JS:
    Prostota: Łatwy do nauki i szybkiego rozpoczęcia pracy.
    Popularność: Miliony przykładów i szeroka społeczność.
    Wydajność: Odciąża serwery, wykonując operacje po stronie klienta.

  7. Framework to:
    rodzaj programu do tworzenia innych programów w JS

  8. Do aplikacji mobilnych można użyć Framework
    Tak, np. React Native do aplikacji na różne systemy (cross-platform).

  9. Aplikacje responsywne to:
    Aplikacje, które dostosowują się do rozmiaru ekranu (przy użyciu CSS, HTML i JS), aby wyglądać dobrze na każdym urządzeniu.

  10. Przepisz linie kody pod nimi wytłumaczenie jak działa ten fragment kodu
    <input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"> Wytłumaczenie: Tworzy przycisk. Po kliknięciu (onclick) wywołuje funkcję JavaScript WinOpen(' '), która prawdopodobnie otwiera nowe okno.

  11. Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200"); Wytłumaczenie: Otwiera nowe okno z stroną "obraz.html", o nazwie "okienko" i konkretnych parametrach:

    toolbar=no ukrywa pasek narzędzi
    directories=no ukrywa przyciski katalogów
    menubar=no ukrywa pasek menu
    height=280 ustawia wysokość okna o wartośći 280px
    width=160 ustawia szerokość okna o wartośći 160px
    top=200 odstęp od góry okna o wartości 200px
    left=200 odstęp od lewej strony okna o wartości 200px
